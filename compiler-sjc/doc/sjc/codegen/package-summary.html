Package sjc.codegen

Provides for StaticJava code generation.

See: Description

  • Class Summary 
    Class Description
    ByteCodeGenerator
    This class is used to translate a StaticJava CompilationUnit to ClassByteCodes that represent a Java 1.5 class files.
    ClassByteCodes
    This class is used to hold the result of the ByteCodeGenerator (when the generation is succesful).
  • Exception Summary 
    Exception Description
    ByteCodeGenerator.Error
    This class is used to signal an error in the process of generating bytecode.
