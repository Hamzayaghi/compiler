Package sjc.annotation

Provides for documentation annotations.

See: Description

  • Annotation Types Summary 
    Annotation Type Description
    NonNull
    An annotation that indicates that a return value of a method, a field (once initialized), or a method parameter is always non-null.
    NonNullElements
    An annotation that indicates both NonNull and NonNullElementsOnly.
    NonNullElementsOnly
    An annotation that indicates that the elements of an array, a collection, or a map of a return value of a method, a field (once initialized), or a method parameter are always non-null.
    PossiblyNull
    An annotation that indicates that a return value of a method, a field (even after initialized), or a method parameter may be null.
    PossiblyNullElements
    An annotation that indicates both PossiblyNull and PossiblyNullElementsOnly.
    PossiblyNullElementsOnly
    An annotation that indicates that the elements of an array, a collection, or a map of a return value of a method, a field (even after initialized), or a method parameter may be null.
    ReadOnly
    An annotation that indicates that field (once initialized) is read-only (write-access is disallowed).
    ReadOnlyElements
    An annotation that indicates both ReadOnly and ReadOnlyElementsOnly.
    ReadOnlyElementsOnly
    An annotation that indicates that the elements of an array, a collection, or a map of a field (once initialized) are read-only (write-access is disallowed).
